# Smart Guess Game (Java)

## 📌 Project Overview

Smart Guess Game is a simple Java console application where the player tries to guess a randomly generated number within a limited number of attempts.

The game offers multiple difficulty levels, calculates the player's score based on remaining attempts, and provides hints after every incorrect guess.

---

## ✨ Features

- Three difficulty levels
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–200)
- Random number generation
- Score calculation
- Limited attempts (10)
- "Too High" and "Too Low" hints
- Displays final result when the game ends

---

## 🛠 Technologies Used

- Java
- Scanner Class
- Random Class
- Console Application


4. Run the program

---

## 🎮 Game Flow

1. Select difficulty level.
2. System generates a random number.
3. Enter your guesses.
4. Receive hints after each attempt.
5. Score is calculated if you guess correctly.
6. Game ends after guessing correctly or exhausting all attempts.
